# Technologies

## Languages
- **C#** - Primary language for Unity development
- **C++** - IL2CPP compilation for mobile builds
- **Shader/HLSL** - Graphics shader programming

## Engines & Frameworks
- **Unity** - Main game engine (2019+)
- **Loxodon.Framework** - UI framework integration
- **FairyGUI** - Advanced UI system for games

## Graphics
- **AssetBundle** - Dynamic resource loading
- **Shader/ParticleSystem** - Visual effects
- **NavMesh** - AI pathfinding
- **LOD (Level of Detail)** - 3D optimization
- **SkinnedMeshRenderer** - Animated 3D models

## Networking
- **SmartFox Server** - Multiplayer game server (TCP/UDP/HTTP)

## Mobile Platforms
- **Android** - Primary platform (ADB, Google Play)
- **iOS** - Secondary platform (App Store)

## Development Tools
- **TeamCity** - CI/CD builds
- **SVN** - Version control
- **ADB** - Android Debug Bridge
- **Maintainer** - Resource optimization tool

## SDKs & Services
- **Firebase** - Analytics & auth

- **Mintegral** - Ad mediation platform

## Build & Optimization
- **IL2CPP** - C++ compilation for performance
- **Unity Profiler** - Performance analysis
- **Static/Dynamic Batching** - Draw call reduction
- **Object Pooling** - Memory optimization

- **SLG** - Strategy game genre
- **3D City/Building** - Internal city system
- **Quest/Inventory** - Game progression systems
