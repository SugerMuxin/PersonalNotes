1.向量的点乘 dot（Vec1,Vec2） 或者  Vec1·Vec2
	反应的是两个向量的相似度，也可以理解为Vec1在Vec2上的投影

2.向量的叉乘 
**叉乘（Cross Product）**又称**向量积**（Vector Product）。

在空间中有两个向量：a = {x1,y1,z1}  ，b = {x2, y2, z2} ， a 与 b之间夹角为  q。

从代数角度计算：
	a x b = (y1z2 - z1y2,z1x2 - x1z2, x1y2 - y1x2)
	

从几何角度计算：(  为  与  所构成平面的单位向量)

其运算结果是一个向量，并且与这两个向量都**垂直**，是这两个向量所在平面的**法线向量**。
